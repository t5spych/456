Проект № 3
Проект реализован с использованием технологии  Flask.
Реализовано использование html-страниц.
Приложение создано как виртуальное меню для кафе. 
Есть переход на несколько страниц, использование дополнительных файлов(изображений),
использованы таблицы в качестве макета.
Приложение локальное, для запуска необходимо использовать локальную ссылку  http://127.0.0.1:8080/ 
Основная программа находится в корнивике с именем flpr3.py
Дополнительный html-код реализован в отдельных файлах(для каждой страницы - свой), находящихся в папке
./templates. Папка ./static/image содержит необходимые для сборки изображения.
